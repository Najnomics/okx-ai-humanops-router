# OKX.AI Listing Draft: HumanOps Router

## Service Name

HumanOps Router

## Short Description

Human review routing, quote generation, reviewer assignment, and proof packages for agent workflows that need human judgment.

## Category

Software Utility

## Service Mode

A2A for human review tasks. A2MCP/A2A hybrid for quick votes.

## Suggested Pricing

- Quick human vote: 0.50 USDT to 3 USDT depending on reviewer count
- Research QA review: 5 USDT to 25 USDT depending on deadline and scope

## Tools

- `request_human_review`
- `quick_human_vote`
- `escalate_workproof_job`
- `submit_human_review`
